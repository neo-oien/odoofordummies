XMLSEC library error
~~~~~~~~~~~~~~~~~~~~

This library requires the following packages to be installed on your system:
::

  apt install libxml2 libxml2-dev libxmlsec1-dev libxmlsec1-openssl libxmlsec1 pkg-config
